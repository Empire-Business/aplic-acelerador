## Hormozi's Voice Characteristics

### Tone:
- **Direct & Definitive** - No hedging, no "maybe"
- **Tough Love** - Brutal honesty, not coddling
- **Action-Oriented** - Focus on doing, not feeling
- **Anti-Excuse** - Calls out rationalizations
- **Practical** - Real-world tested advice

### Style:
- **Short & Punchy** - Many tweets are 1-2 sentences
- **Specific Numbers** - "10 ways", "90 days", "3 things"
- **Visual Formatting** - Line breaks, emojis, arrows
- **Repeatable Phrases** - "Trade X for Y" patterns
- **Casual Language** - "shit", "stuff", but profound content

### Content Patterns:
- **Frameworks** - Loves breaking things into 3-5 steps
- **Binary Choices** - A vs B, pick one
- **Definitions** - "X is actually Y"
- **Lists** - Numbered, scannable
- **Counterintuitive** - Challenges conventional wisdom
- **Personal Stories** - "When I was X, I learned Y"

## How to Use These Files

### For Learning the Voice:
1. **Read each category** to understand different tweet types
2. **Study the analysis** to understand why tweets work
3. **Practice with templates** to create similar content
4. **Mix and match** elements from different categories

### For Creating New Content:
1. **Choose the format** from viral-formats.md
2. **Select the category** (hook, framework, motivation, etc.)
3. **Use the template** as structure
4. **Apply Hormozi's voice** characteristics
5. **Test against checklist** in viral-formats.md

### Key Principles:
- **Truth > Feelings** - Say what's true, not what feels good
- **Action > Theory** - Practical over philosophical
- **Specific > General** - Numbers, examples, concrete steps
- **Definitive > Hedged** - "Always" not "usually"
- **Short > Long** - One powerful idea, well-expressed
