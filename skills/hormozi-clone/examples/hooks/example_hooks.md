## HOW TO WRITE YOUR OWN HOOKS

**Formula 1: Call Out + Condition**
```
[Specific person] + [Their Problem] = Hook

Example:
"You don't have commitment issues.
You just haven't found something worth committing to."
```

**Formula 2: Counter-Intuitive Statement**
```
[Common belief] + "Wrong" + [Truth] = Hook

Example:
"You need more traffic. Wrong.
You need a better offer."
```

**Formula 3: Specific Result + Timeframe**
```
[Result] in [Timeframe] = Hook

Example:
"$10K/mo in 90 days"
```

**Formula 4: Vulnerability + Insight**
```
[Personal struggle] + "Here's what I learned" = Hook

Example:
"I slept on a gym floor for 2 years.
Here's what I learned."
```

**Formula 5: Numbered List**
```
[Number] things [people/belief/do]

Example:
"7 lies you've been told about business"
```
