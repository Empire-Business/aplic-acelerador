## Key Concepts Across All Playbooks

### The Value Equation
```
Value = (Dream Outcome × Perceived Likelihood) ÷ (Time × Effort)
```

This equation appears throughout Hormozi's work and is the foundation for:
- Offer creation
- Pricing strategy
- Retention systems
- LTV optimization

### The 70-20-10 Rule
Borrowed from Google's innovation strategy:
- **70%** - Proven winners (do more of what works)
- **20%** - Winner adjacent (model others' successes)
- **10%** - Radical experiments (moonshots)

Applied to:
- Ad creation
- Hook testing
- Content strategy
- Offer optimization

### Testing Over Opinions
- "Let the data do the teaching"
- Test 10× more than competitors
- Document everything
- Top 10% → replicate
- Bottom 10% → eliminate forever

## Common Metrics Benchmarks

### Conversion Rates
- **Poor:** <2%
- **Okay:** 2-5%
- **Good:** 5-10%
- **Great:** >10%

### Churn Rates (Monthly)
- **Poor:** >15%
- **Okay:** 10-15%
- **Good:** 5-10%
- **Great:** <5%

### LTV:CAC Ratio
- **Break even:** 1:1
- **Viable:** 2:1
- **Healthy:** 3:1
- **Excellent:** 5:1+

### Ad CTR
- **Poor:** <1%
- **Okay:** 1-2%
- **Good:** 2-5%
- **Great:** >5%
