## Frameworks Principais Identificados

### 1. Os Core Four
```
Audiência Quente (Pessoas que conhecem você)
├── 1-para-1: Warm Outreach
└── 1-para-muitos: Postar Conteúdo Gratuito

Audiência Fria (Estranhos)
├── 1-para-1: Cold Outreach
└── 1-para-muitos: Anúncios Pagos
```

### 2. Lead Getters (Captadores de Leads)
```
1. Clientes → Indicações/Boca a Boca
2. Funcionários → Equipe interna
3. Agências → Parceiros externos
4. Afiliados → Parceiros de negócio
```

### 3. Framework Mais-Melhor-Novo
```
Mais: Aumentar volume e frequência
Melhor: Otimizar através de testes
Novo: Expandir para novas plataformas/formatos
```

### 4. Fórmulas de Valor
```
LTGP (Lifetime Gross Profit) = Todo o dinheiro que cliente gasta - Custo de entrega
CAC (Customer Acquisition Cost) = Custo total para adquirir cliente
Ratio Ideal: LTGP:CAC ≥ 3:1
```

### 5. Aquisição Financiada pelo Cliente
```
Dinheiro nos primeiros 30 dias ≥ CAC + Custo de atendimento
= Escala ilimitada através de reinvestimento
```

### 6. Regra dos 100
```
100 ações primárias por dia
× 100 dias consecutivos
= Leads engajados garantidos
```
