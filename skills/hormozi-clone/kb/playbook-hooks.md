## THE HOOK FORMULA

```
HOOK = Call Out + Condition for Value
```

**Call Out:** Makes the prospect say "this is for me"
- Activates the "cocktail party effect" (like hearing your name in a noisy room)
- Examples: "Local business owners...", "If you're struggling with..."

**Condition for Value:** The promise of value IF they consume
- Justifies their time investment
- Examples: "...read this", "...here's why", "...watch this"

---

## THE 8 TYPES OF VERBAL HOOKS

1. **Labels** - "Local business owners..."
2. **Questions** - Yes/No questions or Open questions
3. **Conditionals** - "If you're X, then Y"
4. **Commands** - "Read this if..."
5. **Statements** - "The smartest thing..."
6. **Lists/Steps** - "28 ways to..."
7. **Narratives** - "One day I was..."
8. **Exclamations** - "Ahhhhh..."

## THE 70-20-10 FRAMEWORK (Google's Innovation Rule)

**For Creating Hooks:**

**70% CORE** - Proven Winners
- Hooks already working (yours or others)
- "More of what works"
- Guarantees baseline performance

**20% EMERGING** - Adjacent Variations
- "Winner adjacent"
- Concepts from other niches adapted
- Promising but less proven

**10% BIG IDEAS** - Radical Experiments
- Completely new concepts
- High risk, high return
- Source of breakthrough innovations

## TESTING SYSTEM

**10x Testing Rule:**
For each piece of content, create 10 different hooks.

**Process:**

**Phase 1: Collection (Week 1)**
- Research 50 hooks from top performers in your industry
- Use YouTube/Instagram/Twitter analytics or manual search
- Document in spreadsheet: Name | Hook | Views | Link

**Phase 2: Categorization**
- Classify by type (Question/Statement/Story/etc)
- Identify patterns

**Phase 3: First Implementation**
- Use ALL 50 collected hooks
- Establish baseline data
- Identify top 10% and bottom 10%

**Phase 4: Optimization (Week 2-4)**
- Apply 70-20-10 framework
- 70%: Create variations of top 5 hooks
- 20%: Adapt concepts from other niches
- 10%: Try radical experiments

**Phase 5: Scale (Month 2+)**
- Top hooks become permanent templates
- Create searchable database
- Build SOPs for team

## DECISION TREE

**For Creating New Content:**

- **IF [First time]** → **THEN [Collect 50 external hooks]**
- **IF [Has historical data]** → **THEN [Apply 70-20-10]**
- **IF [Hook in top 10%]** → **THEN [Create 5+ variations]**
- **IF [Hook in bottom 10%]** → **THEN [Never use again]**
- **IF [Average performance]** → **THEN [Test 1-2 variations only]**

**For Choosing Hook Type:**

- **IF [Cold audience]** → **THEN [Questions or Exclamations]**
- **IF [Warm audience]** → **THEN [Conditionals or Stories]**
- **IF [Hot audience]** → **THEN [Commands or Statements]**

**Lagging Indicators (Weekly):**

```

[4-5s]: Transition to content
```

## NINJA HACKS

**Hack #1: The Banana Principle**
- Do something visual unexpected in first 2 seconds
- Pattern interrupt activates involuntary attention

**Hack #2: The Mirror Question**
- "Have you ever [universal but unadmitted situation]?"
- Creates instant private identification

**Hack #3: Real Quick**
- Start with "Real quick question/Real quick..."
- Implies brevity, reduces resistance

**Hack #4: That's Weird**
- Start with observation of something wrong/strange
- Mental inconsistency demands resolution

**Hack #5: The Confession Open**
- "I have a confession..." + controversial truth
- Vulnerability creates trust
