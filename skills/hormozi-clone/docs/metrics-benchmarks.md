Traffic → Lead → Opportunity → Customer

Traffic to Lead: 10-30%

**Optimization Priorities:**

2. Then move to next biggest
3. Test one variable at a time
4. Document results

## RETENTION METRICS

### Churn Rate Benchmarks

**Monthly Churn Rate:**
```
Churn Rate = (Customers Lost ÷ Total Customers) × 100
```

| Rating | Monthly Churn | Annual Churn |
|--------|--------------|--------------|
| **Terrible** | >20% | >90% |
| **Poor** | 15-20% | 80-90% |
| **Okay** | 10-15% | 70-80% |
| **Good** | 5-10% | 45-70% |
| **Great** | 3-5% | 30-45% |
| **Elite** | <3% | <30% |

**The Churn Mathematics:**
```
10% Monthly Churn:
- Average customer lifespan: 10 months
- $100/month × 10 months = $1,000 LTV

3% Monthly Churn:
- Average customer lifespan: 33 months
- $100/month × 33 months = $3,300 LTV

Result: 3.3× increase in LTV from same customer
```

## LTV & CAC METRICS

### Lifetime Value (LTV)

**LTV Formula:**
```
LTV = Average Revenue Per Customer ÷ Churn Rate

Example: $100/month ÷ 5% churn = $2,000 LTV
```

**LTV Benchmarks:**
| Business Type | Typical LTV |
|--------------|-------------|
| **Low-ticket subscription** | $100-500 |
| **Mid-ticket subscription** | $500-2,000 |
| **High-ticket subscription** | $2,000-10,000 |
| **One-time low-ticket** | $50-200 |
| **One-time mid-ticket** | $200-1,000 |
| **One-time high-ticket** | $1,000-10,000 |

**By Industry:**
- **SaaS:** $1,000-10,000
- **E-commerce:** $100-1,000
- **Info Products:** $200-3,000
- **Coaching/Consulting:** $2,000-20,000
- **Services:** $1,000-50,000

---

### Customer Acquisition Cost (CAC)

**CAC Formula:**
```
CAC = Total Sales & Marketing Costs ÷ Number of New Customers

Include: Ad spend, salaries, software, agency fees, etc.
```

**CAC Benchmarks:**
| Business Type | Typical CAC |
|--------------|-------------|
| **Low-ticket** | $10-50 |
| **Mid-ticket** | $50-500 |
| **High-ticket** | $500-5,000 |
| **Ultra-high-ticket** | $2,000-20,000 |

**By Channel:**
- **Warm Referrals:** $0-50
- **Warm Outreach:** $50-200
- **Cold Outreach:** $100-500
- **Content Marketing:** $50-300
- **Paid Ads:** $100-1,000

---

### LTV:CAC Ratio

**The Most Important Metric:**
```
LTV:CAC Ratio = Lifetime Value ÷ Acquisition Cost
```

| Rating | LTV:CAC Ratio | Viability |
|--------|--------------|-----------|
| **Disaster** | <1:1 | Losing money |
| **Break Even** | 1:1 | Not sustainable |
| **Viable** | 2:1 | Barely growing |
| **Healthy** | 3:1 | Sustainable growth |
| **Excellent** | 5:1+ | Rapid scaling |

**The 3:1 Rule:**
- Aim for minimum 3:1 LTV:CAC
- Allows for healthy profit margins
- Enables reinvestment in growth
- Provides buffer for market changes

**Payback Period:**
```
Payback Period = CAC ÷ Monthly Revenue Per Customer

Ideal: <12 months to recover CAC
Excellent: <6 months to recover CAC
```

---

### LTV Optimization Impact

**The Crazy 8 Framework:**
Eight ways to increase customer value:

1. **Raise prices** - Immediate LTV increase
2. **Add upsells** - Increase initial purchase
3. **Add downsells** - Capture lost sales
4. **Add cross-sells** - Expand purchase breadth
5. **Add continuity** - Create recurring revenue
6. **Increase frequency** - More purchases per year
7. **Add referrals** - Lower CAC
8. **Reduce churn** - Extend customer lifespan

**Example Impact:**
```
Starting Point:
- Price: $100/month
- Churn: 10%
- LTV: $1,000

After Optimization:
- Price: $150/month (50% increase)
- Churn: 5% (50% reduction)
- LTV: $3,000 (3× increase)
```

## HOW TO CALCULATE EVERYTHING

### LTV Calculation

**Method 1: Simple (Subscription)**
```
LTV = Average Revenue Per Customer ÷ Churn Rate

Example: $100/month ÷ 5% = $2,000
```

**Method 2: Advanced (All Customers)**
```
LTV = (Average Order Value × Purchase Frequency) × Customer Lifespan

Example: ($150 × 4 purchases/year) × 3 years = $1,800
```

**Method 3: Cohort-Based (Most Accurate)**
```
LTV = Sum of all revenue from cohort ÷ Number of customers in cohort
```

---

### CAC Calculation

**Simple CAC:**
```
CAC = Total Marketing Spend ÷ New Customers
```

**Fully Loaded CAC:**
```
CAC = (Marketing + Sales salaries + Software + Agency fees) ÷ New Customers
```

**Blended CAC:**
```
Blended CAC = Total Acquisition Costs ÷ Total New Customers
(Include both paid and organic)
```

---

### Churn Calculation

**Monthly Churn:**
```
Churn Rate = (Customers Lost This Month ÷ Customers at Start of Month) × 100
```

**Annual Churn:**
```
Annual Churn = (Customers Lost This Year ÷ Customers at Start of Year) × 100
```

**Revenue Churn:**
```
Revenue Churn = (MRR Lost ÷ MRR at Start) × 100
```

---

### Conversion Rate Calculation

**Simple Conversion:**
```
Conversion Rate = (Conversions ÷ Visitors) × 100
```

**Funnel Conversion:**
```
Stage Conversion = (People in Next Stage ÷ People in Current Stage) × 100
```

**Total Funnel Conversion:**
```
Total Conversion = (Customers ÷ Initial Visitors) × 100
```

---

### ROAS Calculation

**Basic ROAS:**
```
ROAS = Revenue from Ads ÷ Ad Spend
```

**ROAS by Campaign:**
```
Campaign ROAS = Campaign Revenue ÷ Campaign Spend
```

**Break-Even ROAS:**
```
Break-Even ROAS = 1 ÷ Profit Margin

Example: 50% margin = 2× break-even ROAS
```

---

### Profit Margin Calculation

**Gross Margin:**
```
Gross Margin = (Revenue - COGS) ÷ Revenue × 100
```

**Net Profit Margin:**
```
Net Profit Margin = (Revenue - All Expenses) ÷ Revenue × 100
```

**Contribution Margin:**
```
Contribution Margin = (Revenue - Variable Costs) ÷ Revenue × 100
```
