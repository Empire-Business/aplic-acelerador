## Os 7 Funis Fundamentais (Brunson Framework)

### 1. Two-Step Tripwire Funnel

**Quando usar:**
- Novo negócio, validando oferta
- Baixo budget de tráfego
- Produto digital entry-level

**Estrutura:**
```
TRÁFEGO
    │
    ▼
┌─────────────────────┐
│ SQUEEZE PAGE        │ → Captura email (opt-in 25-40%)
│ Lead Magnet Grátis  │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ THANK YOU PAGE      │ → Tripwire (3-10% CR)
│ Oferta de entrada   │   R$ 27-197
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ UPSELL #1           │ → Upsell (15-30% take rate)
│ Upgrade/Core        │   R$ 297-997
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ UPSELL #2 / DOWNSELL│ → Segundo upsell ou downsell
└─────────────────────┘
```

**Métricas-chave:**
- Opt-in rate: > 30%
- Tripwire CR: > 5%
- Upsell take rate: > 20%
- AOV alvo: R$ 150+

**Ferramentas recomendadas:**
- ClickFunnels, Leadpages, ou custom
- Stripe/PayPal para pagamentos
- Zapier para automações

### 2. Perfect Webinar Funnel

**Quando usar:**
- Core offer de R$ 997-4.997
- Precisa de educação do mercado
- Quer escalar com afiliados

**Estrutura:**
```
TRÁFEGO (Multi-source)
    │
    ▼
┌─────────────────────┐
│ REGISTRATION PAGE   │ → 20-40% opt-in
│ Webinar gratuito    │   (Captura + lembrete)
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ PRÉ-WEBINAR SEQUENCE│
│ (4 emails + SMS)    │ → Aumenta show-up rate
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ WEBINAR ROOM        │ → Show-up 25-35%
│ 3 Secrets Framework │   Stay until offer 60%+
│ + The Stack         │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ PÓS-WEBINAR         │
│ Replay + Follow-up  │ → 7-day close sequence
└─────────────────────┘
```

**The 3 Secrets Framework:**

**SECRET #1:** "A única coisa que realmente importa"
- Reframe da crença errada #1
- Destrói crença limitante do mercado

**SECRET #2:** "O verdadeiro motivo pelo qual você não consegue"
- Revela a causa raiz do problema
- Posiciona sua solução como única saída

**SECRET #3:** "O atalho que ninguém te contou"
- Sua metodologia/framework único
- Cria desejo intenso pela solução completa

**Métricas-chave:**
- Registrations: Alvo depende de budget
- Show-up rate: > 25%
- Stay until pitch: > 60%
- Conversion: > 5% (ótimo: 10%+)

### 3. Invisible Funnel (VSL)

**Quando usar:**
- Tráfego frio (cold ads)
- Automação total
- Produto de R$ 497-1.997

**Estrutura:**
```
TRÁFEGO PAGO (Frio)
    │
    ▼
┌─────────────────────┐
│ VSL PAGE            │
│ Video Sales Letter  │ → 15-25 min
│ Auto-play           │   Botão aparece após minuto X
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ ORDER FORM          │
│ + Order Bump        │ → Bump: 30-40% take rate
│ + Upsell #1         │   Upsell: 20-30% take rate
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ RECOVERY SEQUENCE   │
│ (7 emails)          │ → Recuperar não-compradores
└─────────────────────┘
```

**VSL Script Framework:**
1. **Hook** (0-2 min): Pattern interrupt + promise
2. **Story/Problem** (2-8 min): Agitar dor + conectar
3. **Solution** (8-15 min): Apresentar sistema
4. **Offer** (15-22 min): The Stack completo
5. **CTA** (22-25 min): Urgência + garantia

### 4. Product Launch Funnel (PLF)

**Quando usar:**
- Lançamento de novo produto
- Buildar lista antes de vender
- Criar buzz e antecipação

**Sequência PLF:**
```
PRÉ-LANÇAMENTO (Semana 1-2)
├── Conteúdo #1: "Oportunidade" (Story)
├── Conteúdo #2: "Transformação" (Change)
└── Conteúdo #3: "Posse" (Ownership)
    └── "Cart aberto amanhã"

LANÇAMENTO (Semana 3 - 5-7 dias aberto)
├── Dia 1: ABERTURA + Fast Action Bonus
├── Dia 2-3: SOCIAL PROOF (Cases)
├── Day 4-5: OBJEÇÕES (FAQ)
├── Day 6: URGÊNCIA máxima
└── Day 7: FECHAMENTO

PÓS-LANÇAMENTO
├── Onboarding compradores
└── Oferta para não-compradores
```

### 5. 7-Day Challenge Funnel

**Quando usar:**
- Engajamento é crucial
- Produto de transformação
- Community building

**Estrutura:**
```
REGISTRO
    │
    ▼
┌─────────────────────┐
│ PÁGINA DE INSCRIÇÃO │ → R$ 27-47 (ou grátis)
│ Desafio 7 Dias      │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ OS 7 DIAS           │
│ Dia 0: Kickoff      │
│ Dia 1-6: Conteúdo   │ → Entrega valor + pequenas vitórias
│ Dia 7: FECHAMENTO   │   Oferta especial participantes
└─────────────────────┘
```

### 6. High-Ticket Application Funnel ⭐

**Quando usar:**
- Ofertas de R$ 10.000-100.000+
- Consultoria/mentoria
- Serviços done-for-you
- QUALIFICAÇÃO é essencial

**Estrutura Completa:**
```
TRÁFEGO QUALIFICADO
    │
    ├── Conteúdo Premium
    ├── Webinar ao vivo
    ├── Indicações
    └── Parcerias (Dream 100)
    │
    ▼
┌─────────────────────┐
│ PÁGINA DE APLICAÇÃO │ → "Vagas limitadas"
│ VSL opcional        │   Investimento: "A partir de R$ XX.XXX"
│ Formulário detalhado│   Formulário qualificador
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ TRIAGEM             │
│ Score A: Aprovado   │ → Agendar call
│ Score B: Potencial  │ → Follow-up
│ Score C: Rejeitado  │ → Nurture sequence
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ DISCOVERY CALL      │ → 30 min
│ (Qualificação)      │   Entender dor + Budget + Timeline
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ SALES CALL          │ → 45-60 min
│ (CCDI + The Stack)  │   Rapport → Discovery → Presentation → Close
└──────────┬──────────┘
           │
           ▼
    ┌──────┴──────┐
    ▼             ▼
┌─────────┐   ┌─────────┐
│ FECHADO │   │FOLLOW-UP│
│+ Setup  │   │Sequence │
└─────────┘   └─────────┘
```

**Critérios de Qualificação (BANT Adaptado):**

| Critério | Pontos | Pergunta-chave |
|----------|--------|----------------|
| Budget | 0-30 | "Qual seu faturamento mensal?" |
| Authority | 0-20 | "Além de você, quem decide?" |
| Need | 0-30 | "Qual seu maior desafio agora?" |
| Timeline | 0-20 | "Quer resolver isso em quanto tempo?" |

**Score:**
- 80+: Aprovado direto
- 60-79: Análise manual
- <60: Não qualificado

### 7. Reverse Squeeze Funnel

**Quando usar:**
- Conteúdo é o produto (YouTubers, podcasters)
- SEO/YouTube strategy
- Autoridade building

**Estrutura:**
```
TRÁFEGO ORGÂNICO
    │
    ├── YouTube
    ├── Podcast
    ├── Blog/SEO
    └── LinkedIn
    │
    ▼
┌─────────────────────┐
│ CONTEÚDO COMPLETO   │ → Entrega valor REAL
│ Gratuito            │   Demonstra expertise
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ SOFT OPT-IN         │ → "Quer o template?"
│ (Download/Bônus)    │   Captura email
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ NURTURE SEQUENCE    │ → Mais valor
│                     │   Ofertas progressivas
└─────────────────────┘
```

## Escolhendo Seu Funnel

```
PERGUNTA 1: Qual seu ticket médio?
├── < R$ 500 → Two-Step Tripwire
├── R$ 500-2.000 → VSL ou Tripwire + Webinar
├── R$ 2.000-10.000 → Perfect Webinar
└── > R$ 10.000 → High-Ticket Application

PERGUNTA 2: Qual seu principal tráfego?
├── Orgânico/SEO → Reverse Squeeze
├── Lista existente → Webinar ou Launch
├── Cold ads → VSL ou Tripwire
└── Indicações/Parcerias → High-Ticket App

PERGUNTA 3: Qual seu recurso mais escasso?
├── Tempo → VSL (automático)
├── Dinheiro → Two-Step Tripwire
├── Credibilidade → Reverse Squeeze
└── Expertise → High-Ticket App
```
