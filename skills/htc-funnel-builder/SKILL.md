## 🧠 Frameworks Principais

### 1. Value Ladder (Brunson)

```
Bait (Grátis) → Tripwire (R$ 27-197) → Core (R$ 997-4.997) → High-Ticket (R$ 10k+) → Continuity
```

Aplicação: `kb/funnel-types.md`

### 2. Value Equation (Hormozi)

```
Value = (Dream Outcome × Perceived Likelihood) ÷ (Time Delay × Effort & Sacrifice)
```

Aplicação: Aumentar numerador, diminuir denominador em toda oferta.

### 3. Perfect Webinar - 3 Secrets (Brunson)

Estrutura de apresentação que converte:
- Secret #1: Quebrar crença errada #1
- Secret #2: Revelar causa raiz
- Secret #3: O atalho (seu método)

Template: `templates/webinar-script-template.md`

### 4. Grand Slam Offer (Hormozi)

```
Oferta = Produto Principal + Pilha de Bônus + Garantia de Risco Zero
```

Aplicação: Criar ofertas incomparáveis e irresistíveis.

### 5. High-Ticket Application Funnel ⭐

```
Tráfego → VSL/Conteúdo → Aplicação → Discovery → Sales Call → Close
```

Para ofertas de R$ 10.000-100.000+. Detalhes em `kb/funnel-types.md`

## 🎯 Como Escolher Seu Funil

```
PERGUNTA 1: Qual seu ticket médio?
├── < R$ 500 → Two-Step Tripwire
├── R$ 500-2.000 → VSL ou Tripwire
├── R$ 2.000-10.000 → Perfect Webinar
└── > R$ 10.000 → High-Ticket Application

PERGUNTA 2: Qual seu tráfego principal?
├── Orgânico/SEO → Reverse Squeeze
├── Lista → Webinar ou Launch
├── Cold ads → VSL ou Tripwire
└── Indicações → High-Ticket App

PERGUNTA 3: Qual recurso é mais escasso?
├── Tempo → VSL (automático)
├── Dinheiro → Two-Step Tripwire
├── Credibilidade → Reverse Squeeze
└── Expertise → High-Ticket App
```
