# Sources Reference

This directory contains references to source materials that informed the Russell Brunson AI Clone skill.

## Primary Sources

### Books by Russell Brunson

1. **DotCom Secrets** (2015)
   - The underground playbook for growing your company online
   - Core concepts: Funnels, Value Ladder, Perfect Webinar
   - Focus: Technical framework of online marketing

2. **Expert Secrets** (2017)
   - Find your voice, build a tribe, and change the world
   - Core concepts: Attractive Character, Epiphany Bridge, Community Building
   - Focus: Positioning, messaging, movement building

3. **Traffic Secrets** (2020)
   - The playbook for filling your funnels and converting leads
   - Core concepts: Dream 100, Traffic sources, Scaling
   - Focus: Audience building and traffic generation

4. **108 Proven Split Test Winners**
   - Collection of successful split tests
   - Focus: Data-driven optimization

5. **The Funnel Hackers Cookbook**
   - 22 funnel recipes with step-by-step instructions
   - Focus: Implementation and templates

6. **30 Days**
   - Russell's personal story and framework
   - Focus: Personal development and entrepreneurship

7. **Marketing Secrets Black Book**
   - Collection of marketing frameworks and strategies
   - Focus: Copywriting and offers

8. **Network Marketing Secrets**
   - Application of funnel principles to MLM
   - Focus: Network marketing industry
