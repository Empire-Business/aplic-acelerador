## Using Specialists

### Direct Invocation

```
/specialist funnel-builder "Design a funnel for a $997 coaching program"
/specialist copywriter "Write a VSL for a weight loss supplement"
/specialist traffic "Plan a Dream 100 strategy for a SaaS startup"
```

### Automatic Selection

The main Russell Brunson skill will automatically invoke the appropriate specialist based on your request.

### Combining Specialists

For complex projects, multiple specialists can collaborate:

```
"I'm launching a new course. Help me with:
1. Funnel design (funnel-builder)
2. Sales copy (copywriter)
3. Launch sequence (launch-manager)
4. Traffic strategy (traffic)"
```

## Specialist Development

### Creating New Specialists

To add a new specialist:

1. Create specialist folder in `/specialists/[name]/`
2. Include `skill.md` with specialist definition
3. Add specialized knowledge in `kb/[specialist-name]/`
4. Create specialist-specific templates in `templates/[specialist-name]/`

### Specialist Template

```markdown
# [Specialist Name] Specialist

## Focus
[Brief description of specialist domain]

## Capabilities
- Capability 1
- Capability 2
- Capability 3

## Use When
- "Example query 1"
- "Example query 2"

## Primary Frameworks
- Framework 1
- Framework 2
- Framework 3

## Related Specialists
- [Other specialist 1]
- [Other specialist 2]
```
