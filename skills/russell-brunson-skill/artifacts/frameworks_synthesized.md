## DotCom Secrets Framework

### Core Philosophy
"Todas as empresas precisam de leads ou clientes. Se você tem um negócio, você precisa de funis."

### The 7 Core Frameworks

#### 1. The Ladder of Value
```
Free Lead Magnet → Tripwire ($7-$47) → Core Offer ($97-$497) →
High Ticket ($997+) → Continuity/Backend
```

**Key Insight**: Stack value, don't swap offers.

#### 2. The Perfect Webinar
```
Secret #1: The Vehicle (one thing away)
Secret #2: Internal Beliefs (false beliefs)
Secret #3: The Stack/Method (external method)
```

**Key Insight**: Stack objections, then stack solutions. Total value should be 10x price.

#### 3. The Epiphany Bridge Story
```
Backstory → Desire → External Conflict → Internal Conflict →
Epiphany → Plan → Conflict → Achievement → Transformation
```

**Key Insight**: Sell new concepts through story, not logic alone.

#### 4. The 23 Building Blocks
```
Pre-Funnel: Lead Magnet, Landing Page
Funnel: Thank You, Sales Page, Order Form, Upsell, Downsell
Follow-Up: Email, SMS, Retargeting
```

**Key Insight**: Funnels are like LEGOs - same blocks, different combinations.

#### 5. Continuity Programs
```
- Create recurring revenue
- Stabilize cash flow
- Build long-term relationships
```

**Key Insight**: Continuity is the foundation of business stability.

#### 6. The Dream 100
```
Identify → Categorize (A/B/C) → Cultivate → Collaborate
```

**Key Insight**: One Dream 100 partnership can 10x your business.

#### 7. Who is Your Dream Customer?
```
- Be specific
- Create avatar
- Target precisely
```

**Key Insight: Better bait = better fish."

## Expert Secrets Framework

### Core Philosophy
"As pessoas não compram produtos; elas compram movimentos/causas/community."

### The Attractive Character Framework

#### Position Options
1. **The Guide** - Been there, solved it, teaching you
2. **The Guru** - Expert authority, respected source
3. **The Relatable** - Friend on the journey with you
4. **The Reporter** - Curating and sharing others' expertise

#### Character Elements
- **Backstory**: Origin story that creates connection
- **Character Flaws**: Vulnerability creates trust
- **Desire**: What you want that they want too
- **Contradictions**: Makes you human and interesting

### The Two Comma Club Framework

#### Qualification
```
Must generate $1M+ in a single funnel
```

#### Categories
- **Award Winners**: Those who achieved it
- **Hall of Fame**: Multiple funnel winners

### The Framework for Teaching

#### Step 1: Find Your Voice
```
- Don't copy others
- Be authentic
- Develop unique style
```

#### Step 2: Tell Your Story
```
- The Epiphany Bridge
- Vulnerability
- Transformation
```

#### Step 3: Systematize Your Knowledge
```
- Create frameworks
- Document processes
- Make teachable
```

#### Step 4: Build Your Community
```
- Create culture
- Identity symbols
- Rituals and traditions
```

## Traffic Secrets Framework

### Core Philosophy
"Traffic is not a problem. Traffic is a skill to be mastered."

### The Three Types of Traffic

#### 1. Owned Traffic
```
- Your email list
- Your audience
- Your community
```

#### 2. Controlled Traffic
```
- SEO (long-term)
- Content marketing
- YouTube/organic social
```

#### 3. Rented Traffic
```
- Facebook Ads
- Google Ads
- Instagram Ads
- Other paid platforms
```

### The Dream 100 Strategy (Detailed)

#### Phase 1: Identification
```
- List 100 influencers in your niche
- Categorize by reach and relevance
```

#### Phase 2: Courting
```
- Start with C-tier (accessible)
- Move to B-tier
- Eventually A-tier
```

#### Phase 3: Tactics
```
- Comment and engage
- Share their content
- Send physical gifts
- Offer value first
- Request collaboration
```

### The Framework for Ad Success

#### Step 1: The Hook
```
- Grab attention in first 3 seconds
- Use curiosity or controversy
```

#### Step 2: The Story
```
- Create emotional connection
- Show transformation
```

#### Step 3: The Offer
```
- Clear value proposition
- Strong call to action
- Urgency/scarcity
```

## Funnel Frameworks

### Core Funnel Types

#### 1. Lead Funnels
```
Purpose: Generate leads
Structure: Opt-in → Thank You → Tripwire Offer
Variations: Squeeze, Reverse Squeeze, Lead Magnet, Survey
```

#### 2. Sales Funnels
```
Purpose: Sell core product
Structure: Sales Page/Video → Order Form → Upsell → Downsell
Variations: VSL, Long-form Sales Letter, Webinar
```

#### 3. Webinar Funnels
```
Purpose: Educate and sell
Structure: Registration → Confirmation → Broadcast/Replay → Offer
Variations: Live, Automated, Hybrid
```

#### 4. Launch Funnels
```
Purpose: Build anticipation and sell
Structure: 3-4 videos over 5-7 days → Open cart → Close cart
Variations: Product Launch, Internal Launch
```

#### 5. Event Funnels
```
Purpose: Sell event registrations
Structure: Sales Page → Registration → Confirmation → Funnels
Variations: Summit, Live Event, Virtual Event
```

### The 7 High-Converting Funnels

#### 1. Two-Step Tripwire
```
Page 1: Low-ticket offer ($7-$47)
Page 2: Order form
Page 3: Thank you + Core offer upsell
```

#### 2. Perfect Webinar
```
Page 1: Webinar registration
Page 2: Confirmation
Page 3: Broadcast room
Page 4: Offer page (special limited time)
```

#### 3. Invisible Funnel
```
Page 1: Video sales letter
Page 2: Order form (inline)
Page 3: Thank you + upsells
```

#### 4. Product Launch
```
Video 1: Opportunity/Mindset
Video 2: The Vehicle
Video 3: The Method
Video 4: The Stack/Offer
```

#### 5. 7-Day Challenge
```
Daily: Training + Action steps
End: Pitch core offer
```

#### 6. High-Ticket Application
```
Page 1: VSL about opportunity
Page 2: Application form
Page 3: Thank you + Call booking
```

#### 7. Reverse Squeeze
```
Page 1: Value video (must watch to proceed)
Page 2: Unlock content (opt-in)
Page 3: Thank you + Offer
```

## Sales Frameworks

### The Perfect Webinar Script

#### Introduction (5 min)
```
- Welcome and housekeeping
- Who is this for?
- What you'll learn
- Why listen to me?
```

#### Content (45 min)
```
Secret #1: The Vehicle (10 min)
- Teaching + Story
- False belief revealed

Secret #2: Internal Beliefs (15 min)
- Teaching + Story
- 3-4 false beliefs addressed

Secret #3: The Method (20 min)
- Step-by-step framework
- Demonstration
- Proof elements
```

#### Close (20 min)
```
- The Stack (value breakdown)
- The Offer
- The Guarantee
- Scarcity/Urgency
- Call to action
```

### The One-Liner Framework
```
Formula: "I help [TARGET MARKET] to [DESIRED RESULT] through [UNIQUE MECHANISM]."

Example: "I help entrepreneurs to build 7-figure businesses through sales funnels."
```

### The Offer Stack Framework
```
1. Core Product: $X (value: $2X)
2. Bonus 1: $Y (value: $3Y)
3. Bonus 2: $Z (value: $3Z)
...
Total Value: $Total
Actual Price: $Price
```

**Key Principle**: Total value should be 10x actual price for irresistible offer.

## Launch Frameworks

#### Week 1: Prep
```

- Prepare bonuses
```

#### Week 2: Soft Launch
```

```

- Webinar tour

- Special pricing
