# Template: The Stack (Apresentação de Valor)

> **"The Stack is about revealing value piece by piece until the price seems insignificant"** — Russell Brunson

---

## Estrutura Completa do Stack

```
┌──────────────────────────────────────────────────────────────┐
│  1. RECAPITULAR → Lembrar onde estavam / onde querem chegar  │
│  2. APRESENTAR VALOR → Componente por componente             │
│  3. REVELAR PREÇO → Com âncora e contraste                   │
│  4. STACKING → Adicionar bônus em cima                        │
│  5. CALL TO ACTION → Assumir a venda                          │
└──────────────────────────────────────────────────────────────┘
```

---

## FASE 1: RECAPITULAÇÃO (2 min)

```
"[Nome], vamos recapitular o que a gente descobriu aqui hoje...

Você está em [situação atual].
Isso está te custando [custo financeiro/emocional] por mês.
Em 12 meses, isso é [projeção].

Você quer chegar em [situação desejada].
Você quer isso em [tempo].
E você disse que está disposto a fazer o que for necessário.

Correto?"

[CONFIRME]

"Perfeito. Então deixa eu te mostrar como a gente vai fazer isso acontecer..."
```

## Checklist do Stack

- [ ] Recap feito (prospect confirma situação)
- [ ] Cada componente tem valor atribuído
- [ ] Soma é significativamente maior que preço
- [ ] Bônus são relevantes para objeções
- [ ] Garantia mencionada
- [ ] Urgência real (fast action bonus)
- [ ] Assumptive close feito
- [ ] Silêncio mantido após ask
