## 🧠 Frameworks Principais

### 1. CCDI Framework ⭐ (Hormozi)

Estrutura completa da sales call (45-60 min):

```
CONNECT (0-10 min)     → Rapport + Contexto
CHALLENGE (10-25 min)  → Quebrar crenças + Gap analysis  
DISCOVER (25-45 min)   → Dor intensa + Desejo profundo
INTENSIFY (45-55 min)  → Urgência + Solução única
CLOSE (55-60 min)      → The Stack + Assumir a venda
```

Detalhado em: `kb/frameworks/ccdi-framework.md`

### 2. The Stack (Brunson)

Apresentação de valor que maximiza percepção:

```
1. RECAPITULAR → Lembrar onde estão
2. APRESENTAR VALOR → Componente por componente
3. REVELAR PREÇO → Com âncora e contraste
4. STACKING → Adicionar bônus
5. CALL TO ACTION → Assumir a venda
```

Template: `templates/the-stack-template.md`

### 3. 5 Levels of Why (Hormozi)

Descoberta de dor profunda:

```
Pergunta 1: "O que está te frustrando?"
Pergunta 2: "Por que isso é problema?"
Pergunta 3: "O que significa isso?"
Pergunta 4: "Como afeta sua vida?"
Pergunta 5: "O que isso diz sobre você?"
```

### 4. Assumptive Close (Hormozi)

Eliminar atrito da decisão:

```
❌ "Você quer comprar?"
✅ "Qual cartão você quer usar?"

❌ "Você vai entrar?"
✅ "Você prefere começar segunda ou terça?"
```

### 5. Top Objection Handlers

**"Preciso pensar"**
→ "O que especificamente você precisa pensar?"

**"Está caro"**
→ Comparar custo de inação

**"Preciso falar com alguém"**
→ Oferecer script para a conversa

**"Não tenho tempo"**
→ "Você não tem tempo, ou não é prioridade?"

Mais 20+ objeções: `kb/objections/top-20-objections.md`
