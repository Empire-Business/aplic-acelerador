## COMO ADAPTAR

1. **Identifique seu resultado principal**: O que seu cliente quer?
2. **Adicione tempo/prazo**: Quanto tempo leva?
3. **Remova objeções**: "Sem [dor comum]"
4. **Seja específico**: Números > palavras genéricas
5. **Teste 3 variações**: Nunca use só uma headline
