# Frameworks: Alex Hormozi para Landing Pages

Síntese dos principais frameworks de Alex Hormozi aplicados a landing pages.

---

## 1. THE VALUE EQUATION (Equação do Valor)

### A Fórmula Matemática da Persuasão

```
         Dream Outcome × Perceived Likelihood of Achievement
VALUE = ─────────────────────────────────────────────────────
                    Time Delay × Effort & Sacrifice
```

### Como Aplcar em Cada Elemento

| Elemento | Para AUMENTAR Valor | Na Landing Page |
|----------|---------------------|-----------------|
| **Dream Outcome** | Promessa maior | Headline ousada |
| **Perceived Likelihood** | Mais prova | Depoimentos, cases |
| **Time Delay** | Resultado mais rápido | "Em X dias" |
| **Effort & Sacrifice** | Menos esforço | "Sem fazer X" |

### Exemplo de Aplicação

**Antes (Value baixo):**
> "Curso de vendas para consultores"

**Depois (Value alto):**
> "Como conseguir 5 clientes novos em 30 dias — mesmo se você não tem experiência, lista ou tempo"

Análise:
- Dream Outcome: 5 clientes novos
- Perceived Likelihood: "mesmo se não tem experiência"
- Time Delay: 30 dias
- Effort & Sacrifice: "sem lista ou tempo"

## 2. GRAND SLAM OFFER (Oferta de Grand Slam)

### Os 4 Elementos de uma Oferta Irresistível

```
┌─────────────────────────────────────────────────────┐
│                 GRAND SLAM OFFER                     │
├─────────────────────────────────────────────────────┤
│                                                     │
│  1. GRAND SLAM PROBLEM                              │
│     Resolve TODOS os problemas (não alguns)        │
│                                                     │
│  2. GRAND SLAM SOLUTION                             │
│     Incomparável (não tem concorrente)             │
│                                                     │
│  3. GRAND SLAM VALUE                                │
│     Valor percebido >> Preço                       │
│                                                     │
│  4. GRAND SLAM DELIVERY                             │
│     Entrega excepcional + experiência              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Checklist de Grand Slam

- [ ] Resolve todos os problemas do cliente?
- [ ] É impossível comparar com concorrentes?
- [ ] Valor percebido é 10x o preço?
- [ ] Experiência de entrega é excepcional?

## 3. GMMM FRAMEWORK (Get-More-Max Money)

### Sequência de Monetização

```
GET MONEY (Atração)
├── Free content (atrai)
├── Lead magnet (captura)
└── Tripwire (primeira compra)

MORE MONEY (Expansão)
├── Core offer (venda principal)
├── Upsells (ofertas adicionais)
└── Bundles (pacotes)

MAX MONEY (Maximização)
├── Continuity (assinatura)
├── High ticket (oferta premium)
└── Backend (produtos adicionais)
```

### Aplicação em Landing Page

```
Página 1: Lead Magnet (GET)
    ↓
Página 2: Tripwire $27 (GET → MORE)
    ↓
Página 3: Core Offer $297 (MORE)
    ↓
Página 4: Continuity $97/mês (MAX)
```

## 4. THE CORE FOUR (Geração de Leads)

### 4 Canais de Aquisição

| Canal | Custo | Tempo | Controle |
|-------|-------|-------|----------|
| **Warm Outreach** | $0 | Rápido | Alto |
| **Content (Post/Video)** | $0 | Médio | Alto |
| **Cold Outreach** | $0 | Médio | Médio |
| **Paid Ads** | $$$ | Imediato | Baixo |

### Para Landing Pages

- **Warm Outreach**: Link direto para página
- **Content**: CTA em posts, bio, descrição
- **Cold Outreach**: DM/email com link
- **Paid Ads**: Campanhas direcionadas

## 5. 70-20-10 TESTING RULE

### Alocação de Testes

```
70% → PROVEN WINNERS
      O que já funciona, continue fazendo
      Headlines que convertem
      CTAs testados

20% → ADJACENT VARIATIONS
      Pequenas mudanças no que funciona
      Trocar palavras, cores, ordem

10% → RADICAL EXPERIMENTS
      Ideias completamente novas
      Arriscar, inovar
```

### Aplicação em Landing Page

- 70% do tráfego na página vencedora
- 20% testando variações pequenas
- 10% testando páginas radicalmente diferentes

## 6. THE HOOK FRAMEWORK

### Fórmula de Hooks que Param o Scroll

```
HOOK = [Contraste] + [Curiosidade] + [Credibilidade]
```

### Tipos de Hooks

**1. Contrarian Hook**
> "Por que [crença comum] está errado"

**2. Story Hook**
> "Como fui de [pior] para [melhor]"

**3. List Hook**
> "[Número] coisas que [grupo] faz diferente"

**4. Question Hook**
> "E se [premissa desafiadora]?"

**5. Promise Hook**
> "Como [resultado] em [tempo] sem [dor]"

## 7. PRICING PSYCHOLOGY

### Ancoragem de Preço

```
VALOR ANCORADO          PREÇO REAL
R$ 2.997         →      R$ 997
   (100%)                  (33%)

"Você economiza R$ 2.000 (67% de desconto)"
```

### Estrutura de Preço em Landing Page

```
┌─────────────────────────────────────┐
│  VALOR TOTAL: R$ 4.997              │
│                                     │
│  ─────────────────────────────────  │
│                                     │
│  Hoje, por apenas:                  │
│                                     │
│  R$ 997                             │
│  ou 12x de R$ 97                    │
│                                     │
│  Economia de R$ 4.000 (80%)         │
└─────────────────────────────────────┘
```

### Regras de Preço

1. **Ancore alto, venda baixo**
2. **Mostre o desconto explicitamente**
3. **Ofereça parcelamento**
4. **Compare com custo da inação**

## 8. RISK REVERSAL (Reverso de Risco)

### Tipos de Garantia

| Tipo | Descrição | Uso |
|------|-----------|-----|
| **Money-back** | Devolve 100% | Digital, cursos |
| **Performance** | Paga se funcionar | Serviços |
| **Anti-guarantee** | Sem garantia (premium) | High ticket |
| **Conditional** | Se cumprir requisitos | Mentorias |

### Fórmula de Garantia

```
[PERÍODO] de garantia incondicional.

Se em [PERÍODO] você não [RESULTADO],
devolvemos 100% do seu dinheiro.
Sem perguntas. Sem burocracia.
O risco é TODO nosso.
```

## 9. SCARCITY & URGENCY (ÉTICO)

### Tipos de Escassez REAL

1. **Quantidade limitada**: "Apenas 50 vagas"
2. **Tempo limitado**: "Fecha em 48 horas"
3. **Bônus limitado**: "Primeiros 30 levam bônus"
4. **Preço limitado**: "Preço sobe na segunda"

### ⚠️ REGRAS ÉTICAS

- ✅ Use escassez REAL
- ✅ Use urgência REAL
- ❌ NUNCA use contagem regressiva falsa
- ❌ NUNCA invente "últimas vagas"
- ❌ NUNCA estenda prazo escondido

## 10. THE DELIVERY MATRIX

### Como Estruturar a Entrega

```
              ALTA SATISFAÇÃO
                   │
         Q4        │        Q1
    (Mantenha)     │    (Promova)
                   │
    BAIXO ESFORÇO ─┼─ ALTO ESFORÇO
                   │
         Q3        │        Q2
    (Automatize)   │    (Elimine)
                   │
              BAIXA SATISFAÇÃO
```

### Aplicação

- **Q1 (Alto esforço, alta satisfação)**: Mantenha e promova
- **Q2 (Alto esforço, baixa satisfação)**: Elimine
- **Q3 (Baixo esforço, baixa satisfação)**: Automatize
- **Q4 (Baixo esforço, alta satisfação)**: Escala
