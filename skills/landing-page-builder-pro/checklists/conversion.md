# Checklist de Conversão

Otimização contínua para aumentar a taxa de conversão.

---

## TESTE DOS 5 SEGUNDOS

Mostre sua página por 5 segundos para alguém que não conhece.

**Perguntas:**
1. O que essa empresa/pessoa vende?
2. Para quem é?
3. O que você ganha?
4. O que você precisa fazer?

**Se não responder as 4 em 5 segundos = REFAÇA**

---

## ABOVE THE FOLD (sem scroll)

Elementos obrigatórios:

- [ ] Headline com promessa clara
- [ ] Subheadline explicando "como"
- [ ] CTA principal (botão)
- [ ] Imagem do produto/resultado OU foto do mentor
- [ ] Elemento de credibilidade (logo de clientes, "as seen on", etc)

**Regra**: O visitante deve entender E poder agir sem scroll.

## PSICOLOGIA DA CONVERSÃO

### Gatilhos Mentais (use com ética)

| Gatilho | Como aplicar |
|---------|--------------|
| **Escassez** | Vagas limitadas REAIS |
| **Urgência** | Prazo real de encerramento |
| **Prova social** | Depoimentos, números, cases |
| **Autoridade** | Credenciais, mídia, experiência |
| **Reciprocidade** | Entregue valor antes de pedir |
| **Compromisso** | Micro-compromissos antes do grande |
| **Afinidade** | Conecte-se com a dor/situação |
| **Contraste** | Mostre antes/depois |

### Evite (Dark Patterns)

- [ ] NUNCA use contagem regressiva falsa
- [ ] NUNCA invente "vagas limitadas" se não é real
- [ ] NUNCA crie urgência artificial
- [ ] NUNCA esconda o preço
- [ ] NUNCA dificulte cancelamento

## OTIMIZAÇÃO DE CTAs

### Texto do Botão

**RUIM:**
- Clique aqui
- Saiba mais
- Enviar
- Submit

**BOM:**
- Quero começar agora
- Garantir minha vaga
- Baixar ebook gratuito
- Começar transformação

**MELHOR:**
- Quero [RESULTADO] em [PRAZO]
- Garantir minha vaga por [PREÇO]
- Baixar [NOME DO MATERIAL] grátis
- Começar [BENEFÍCIO ESPECÍFICO]

### Posicionamento

- [ ] 1 CTA above the fold
- [ ] CTA após cada seção de benefícios
- [ ] CTA após cada seção de prova
- [ ] CTA "sticky" no mobile (fixo no rodapé)
- [ ] Mínimo 6 CTAs na página inteira

### Cor do Botão

- [ ] Contrasta com o fundo
- [ ] Diferente de outras cores da página
- [ ] Testado A/B (vermelho vs verde vs laranja)

---

## FORMULÁRIOS

### Campos Mínimos vs Completos

**Lead Magnet:** Nome + Email (2 campos)
**Webinar:** Nome + Email + Telefone (3 campos)
**High Ticket:** Nome + Email + Telefone + Respostas (5-7 campos)

### Regras

- [ ] Apenas campos necessários
- [ ] Labels claros (não só placeholders)
- [ ] Validação em tempo real
- [ ] Mensagem de erro específica
- [ ] Botão com texto de ação (não "Enviar")
- [ ] Confirmação clara após submissão

---

## PROVA SOCIAL

### Tipos (use pelo menos 3)

1. **Depoimentos escritos** + foto + nome + cargo
2. **Depoimentos em vídeo** (mais confiável)
3. **Cases/Resultados** com números específicos
4. **Logos de clientes** ("Confiam na gente")
5. **Números totais** ("10.000+ alunos")
6. **Avaliações** (estrelas, Trustpilot)
7. **Mídias** ("Visto em Folha, UOL...")

### Posicionamento

- [ ] Primeira prova social aparece antes do primeiro CTA secundário
- [ ] Prova específica para cada objeção
- [ ] Seção dedicada de depoimentos
- [ ] Prova social no CTA final

---

## VELOCIDADE E PERFORMANCE

### Metas

- [ ] First Contentful Paint < 1.5s
- [ ] Largest Contentful Paint < 2.5s
- [ ] Time to Interactive < 3.5s
- [ ] Cumulative Layout Shift < 0.1

### Otimizações

- [ ] Imagens em WebP
- [ ] Lazy loading para imagens abaixo do fold
- [ ] CSS crítico inline
- [ ] JavaScript defer/async
- [ ] CDN para assets
- [ ] Cache configurado

**Cada 1s de carregamento = -7% conversão**

---

## TESTES A/B PRIORITÁRIOS

### Alto Impacto

1. **Headline** - Teste 3 variações
2. **CTA (texto)** - Teste 3 variações
3. **CTA (cor)** - Teste 3 cores
4. **Hero image** - Teste 2 variações
5. **Preço** - Teste anchor diferente

### Médio Impacto

6. **Estrutura da oferta** - Lista vs cards
7. **Depoimentos** - Escritos vs vídeo
8. **Garantia** - Texto curto vs longo
9. **FAQ** - Com vs sem
10. **Formulário** - 2 campos vs 3 campos

### Regra 70-20-10 (Hormozi)

- 70% do tráfego em versão vencedora
- 20% em variações adjacentes
- 10% em experimentos radicais

## MÉTRICAS PARA ACOMPANHAR

### Principais

| Métrica | Meta | Como melhororar |
|---------|------|-----------------|
| Taxa de conversão | > 5% | Copy, CTA, oferta |
| Tempo na página | > 3 min | Conteúdo, engajamento |
| Bounce rate | < 50% | Relevância, velocidade |
| Scroll depth | > 75% | Conteúdo, design |

### Funil

```
Visitantes → Views → Clicks no CTA → Form submits → Compras
   100%        80%       15%            8%          3%

Identifique onde perde mais gente e otimize ali.
```
