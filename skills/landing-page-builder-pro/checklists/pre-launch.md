# Checklist Pré-Lançamento

Use este checklist antes de publicar qualquer landing page.

---

## COPY

- [ ] **Headline**: Tem benefício claro + especificidade + gancho?
- [ ] **Subheadline**: Explica o mecanismo e qualifica o público?
- [ ] **Lead**: Cria conexão emocional nos primeiros 3 segundos?
- [ ] **Problema**: Descreve a dor de forma visceral?
- [ ] **Solução**: Apresenta o mecanismo único claramente?
- [ ] **Benefícios**: Lista 5-7 benefícios específicos?
- [ ] **Prova**: Inclui depoimentos, cases ou números?
- [ ] **Oferta**: Stack de valor claro com preço ancorado?
- [ ] **Garantia**: Remove o risco da compra?
- [ ] **Objeções**: Responde as 3 principais objeções?
- [ ] **CTAs**: Aparece 6+ vezes com texto de ação claro?
- [ ] **Urgência**: É REAL (não falsa contagem regressiva)?
- [ ] **FAQ**: Tem 10-15 perguntas com respostas honestas?

---

## DESIGN

- [ ] **Hierarquia**: H1 > H2 > H3 > p claro visualmente?
- [ ] **CTA Principal**: Visível sem scroll (above the fold)?
- [ ] **Contraste**: CTA se destaca do resto da página?
- [ ] **Cores**: Máximo 3-4 cores com propósito definido?
- [ ] **Tipografia**: 2 famílias de fonte (display + body)?
- [ ] **Whitespace**: Generoso, não apertado?
- [ ] **Imagens**: Alta qualidade, relevantes, otimizadas?
- [ ] **Ícones**: Consistentes no estilo?
- [ ] **Animações**: Têm propósito, não são decoração?
- [ ] **Mobile**: Testado em diferentes tamanhos de tela?

---

## TÉCNICO

- [ ] **Responsivo**: Funciona em mobile (60%+ do tráfego)?
- [ ] **Velocidade**: Carrega em < 3 segundos?
- [ ] **Fontes**: Carregadas com display:swap?
- [ ] **Imagens**: Comprimidas (WebP quando possível)?
- [ ] **Formulário**: Funciona e valida corretamente?
- [ ] **Integração**: Conectado ao email marketing/CRM?
- [ ] **Analytics**: GTM/GA4 configurado?
- [ ] **Pixel**: Facebook/Google/TikTok instalado?
- [ ] **SSL**: HTTPS funcionando?
- [ ] **Favicon**: Configurado?
- [ ] **Meta tags**: Title e description otimizados?

---

## SEO

- [ ] **Title tag**: < 60 caracteres com keyword?
- [ ] **Meta description**: < 155 caracteres persuasivo?
- [ ] **H1**: Único e com keyword principal?
- [ ] **Alt text**: Em todas as imagens?
- [ ] **URL**: Curta e descritiva?
- [ ] **Schema markup**: Implementado (opcional)?

---

## CONVERSÃO

- [ ] **Teste dos 5 segundos**: Dá pra entender o que é?
- [ ] **Caminho único**: Fica claro o que fazer?
- [ ] **Distrações**: Removidas navegação e links desnecessários?
- [ ] **Prova social**: Aparece cedo na página?
- [ ] **Risco removido**: Garantia clara e visível?
- [ ] **Escassez**: É verdadeira e visível?

---

## LEGAL (quando aplicável)

- [ ] **LGPD/GDPR**: Checkbox de aceite de termos?
- [ ] **Política de privacidade**: Link visível?
- [ ] **Termos de uso**: Link visível?
- [ ] **Disclaimers**: Para saúde/finanças quando necessário?
- [ ] **Contato**: Informações de contato visíveis?

---

## PÓS-PUBLICAÇÃO

- [ ] **Teste de formulário**: Submeteu e recebeu email?
- [ ] **Teste de pagamento**: Fluxo completo funciona?
- [ ] **Teste mobile**: Testou em celular real?
- [ ] **Page speed**: Testou no PageSpeed Insights?
- [ ] **Cross-browser**: Testou em Chrome, Safari, Firefox?
- [ ] **Heatmap**: Hotjar/CrazyEgg instalado?
- [ ] **Backup**: Tem cópia do código?

---

## CHECKLIST RÁPIDO (30 segundos)

Antes de publicar, responda:

1. **Em 5 segundos**, alguém entende o que você vende?
2. **O CTA** está visível sem scroll?
3. **A oferta** é clara e irresistível?
4. **A prova** social aparece?
5. **O formulário** funciona?

Se respondeu NÃO para qualquer um, NÃO publique ainda.
