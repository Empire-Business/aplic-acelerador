# 1. VISÃO GERAL DA CAMPANHA

## 📌 Objetivo da Campanha

| Métrica | Meta |
|---------|------|
| **Objetivo Principal** | Geração de leads qualificados (agendamentos de diagnóstico) |
| **Funil** | Top of Funnel (TOFU) → Middle of Funnel (MOFU) |
| **Métrica Principal** | Custo por Lead Qualificado (CPLQ) < R$ 150 |
| **Métrica Secundária** | Taxa de conversão landing page > 15% |
| **Retargeting** | Engajamento 95% + visitantes site |

## 🎯 Público-Alvo Detalhado

### Avatar Principal: "O Empresário Invisível"

| Característica | Descrição |
|----------------|-----------|
| **Faixa Etária** | 35-50 anos |
| **Faturamento** | R$ 400k - R$ 5M/ano |
| **Segmento** | Venda de conhecimento (mentorias, consultorias, cursos B2B, serviços especializados) |
| **Localização** | Grandes centros urbanos (SP, RJ, BH, POA, Curitiba, Brasília) |
| **Time** | Já tem time de conteúdo (2-5 pessoas) |

### Psychographics

**Dores Principais:**
- 😰 **Time não entrega** - Contratou, pagou caro, mas o conteúdo não sai
- 💸 **Custo de tráfego altíssimo** - Gasta R$ 5.000 para vender R$ 10.000
- 😳 **Vergonha do Instagram** - Quando pedem o Instagram em eventos, quer sumir
- 🙈 **Invisibilidade** - É bom no que faz, mas o mercado não conhece

**Desejos:**
- 📈 Gerar mais demanda de forma previsível
- 💰 Aumentar conversão sem aumentar ad spend
- 🎤 Ter mais autoridade no mercado
- ⏰ Liberdade de tempo (não gravar conteúdo todo dia)

### Objeções
- "Já tentei conteúdo e não funcionou"
- "Meu produto é diferenciado, não dá para escalar assim"
- "Preciso de mais time, não de mais estratégia"
- "Isso funciona para influencer, não para empresa B2B"

## 💎 Proposta de Valor Resumida

> Transforme **1 hora de conteúdo** em **50+ peças distribuíveis** — sem precisar gravar reels ou virar "personagem" das redes sociais.

**Promessas Principais:**
1. 🔥 **Mais demanda** - Sistema de geração de demanda previsível
2. 📊 **Maior conversão** - Reduza o CAC com conteúdo de alta conversão
3. 🏆 **Mais autoridade** - Disponibilidade mental no seu mercado

## 🎨 Tom de Voz Visual

| Atributo | Descrição | Evitar |
|----------|-----------|--------|
| **Autoridade** | Dados científicos, cases reais, linguagem precisa | Falsidade, arrogância |
| **Empatia** | Entende a dor do empresário, fala da vergonha | Paternalismo, condescendência |
| **Provocação** | Questiona o óbvio, desafia o "postar a qualquer custo" | Agressividade, ofensa |
| **Esperança** | Mostra que existe solução, é possível escalar | Falsas promessas, clickbait |
| **Profissionalismo** | Design limpo, cores sóbrias, sem firula | Amadorismo, poluição visual |

# 3. CONCEITOS CRIATIVOS

## 🎯 CONCEITO 1: A Dor do Custo de Tráfego

### Ângulo Criativo
> "Você está gastando muito dinheiro com tráfego? Tem empresário que para fazer uma venda de 10 mil ele gasta 5 de tráfego"

### Elementos Visuais Sugeridos
- 📊 Gráfico de barras mostrando "Custo de Aquisição vs Ticket Médio"
- 💸 Representação visual de dinheiro "queimando" ou escapando
- 📉 Seta para baixo indicando ROI negativo
- 🔄 Comparação lado a lado: "Antes" (CAC alto) vs "Depois" (CAC reduzido)

### Textos para Arte

| Tipo | Sugestões |
|------|-----------|
| **Headline Principal** | "Você gasta R$5.000 para vender R$10.000?" |
| | "Seu tráfego está comendo sua margem?" |
| | "CAC alto não é normal" |
| **Subheadline** | "Existe uma forma de reduzir seu custo de aquisição em 40%" |
| | "Empresários de R$400k+ estão trocando tráfego pago por demanda orgânica" |
| **CTA Visual** | "Descubra como →" / "Quero reduzir meu CAC" |

### Tom de Voz
**Alarmante mas Esperançoso**
- Começa com o problema (dói reconhecer)
- Mostra que não é culpa do empresário
- Oferece uma saída real

### Variações
1. **Versão Gráfico**: Mostrar matemática do problema (R$5k ads = R$10k venda = margem zero)
2. **Versão Metáfora**: Torneira aberta, dinheiro escoando
3. **Versão Testimonial**: Print de resultado "Reduzi meu CAC de 60% para 25%"

3. **Versão Split**: Antes (vergonha) / Depois (orgulho)

## 🎬 CONCEITO 3: O Paradoxo do Reel

### Ângulo Criativo
> "Gravar uma aula é mais fácil que gravar um reel. E se você pudesse escalar conteúdo sem gravar reels?"

### Elementos Visuais Sugeridos
- 🎓 Aula/aulão em ambiente profissional (fácil, natural)
- 📱 Tentativa de gravar reel (forçado, constrangedor)
- ⚖️ Balança comparando esforço vs resultado
- 🔄 Seta: 1 aula → 50 peças de conteúdo

### Textos para Arte

| Tipo | Sugestões |
|------|-----------|
| **Headline Principal** | "Gravar aula é fácil. Reel é um parto." |
| | "Você não precisa gravar reels para escalar conteúdo" |
| | "1 aula = 50 peças. Sem gravar reel." |
| **Subheadline** | "Empresários de R$400k+ não gravam reels. Eles gravam aulas." |
| | "Transforme sua expertise natural em máquina de conteúdo" |
| **CTA Visual** | "Quero escalar sem reels" / "Mostre-me como" |

### Tom de Voz
**Provocativo mas Libertador**
- Desafia o "conselho comum" das agências
- Liberta do fardo do "virar influencer"
- Empodera o empresário a ser ele mesmo

### Variações
1. **Versão Comparação**: Lado a lado "Gravando reel" vs "Dando aula"
2. **Versão Processo**: Esteira industrial: aula entra → 50 peças saem
3. **Versão Contrarian**: "Agências odeiam este segredo"

## 🎨 Paleta de Cores Recomendada

### Cores Primárias

| Cor | HEX | Uso |
|-----|-----|-----|
| **Azul Profundo** | `#0A2540` | Fundos escuros, autoridade |
| **Azul Elétrico** | `#00D4AA` | CTAs, acentos, elementos interativos |
| **Branco** | `#FFFFFF` | Textos em fundo escuro, espaço em branco |

### Cores Secundárias

| Cor | HEX | Uso |
|-----|-----|-----|
| **Laranja Alerta** | `#FF6B35` | Destaque de dores, urgência |
| **Cinza Neutro** | `#6B7280` | Textos secundários, elementos sutis |
| **Creme** | `#F5F5F0` | Fundos claros, alternativa ao branco puro |

### Gradients Sugeridos
- Fundo escuro: `linear-gradient(135deg, #0A2540 0%, #1a365d 100%)`
- CTA: `linear-gradient(90deg, #00D4AA 0%, #00B894 100%)`
- Destaque: `linear-gradient(135deg, #FF6B35 0%, #FF8C61 100%)`

## ✨ Estética Recomendada

### DOs ✅
- Design clean, minimalista, com muito espaço em branco
- Fotografia autêntica (não stock genérico)
- Tipografia moderna sans-serif
- Contraste alto para legibilidade mobile
- Elementos que "respiram" (não poluir)
- Hierarquia visual clara (o que olhar primeiro)
- Consistência entre peças (família visual)

### DON'Ts ❌
- Poluição visual (muitos elementos)
- Cores chamativas excessivas (parece golpe)
- Fontes decorativas ou difíceis de ler
- Fotos de stock óbvias (modelos sorrindo demais)
- Texto pequeno demais para mobile
- Gradients excessivos ou mal aplicados
- Inconsistência visual entre anúncios

## ⚠️ O que Evitar em Criativos

| Erro | Por que Evitar |
|------|----------------|
| **Antes/Depois de peso/corpo** | Parece fitness/nutri, confunde o público |
| **Promessas de renda fácil** | Associado a golpe, quebra a confiança |
| **Imagens de dinheiro voando** | Vulgar, atrava publicidade |
| **Texto em excesso** | Ninguém lê, algoritmo penaliza |
| **Clickbait agressivo** | Alta rejeição, custo aumenta |
| **Depoimentos genéricos** | "Mudou minha vida" sem contexto = fake |
| **Cores fluorescentes** | Parece anúncio de trading/cassino |

# 6. BRIEF POR FORMATO

## 📱 Brief - Instagram Reels (9:16)

| Elemento | Especificação |
|----------|---------------|
| **Duração** | 30-45 segundos |
| **Hook (0-3s)** | Visual impactante + problema na cara |
| **Estrutura** | Hook → Problema → Solução → Prova → CTA |
| **Áudio** | Trending sound ou voz over própria |
| **Legendas** | Sempre ativas, tamanho grande |
| **Safe Zone** | Texto centralizado, 250px das bordas |
| **Thumbnail** | Face expressiva + 3-5 palavras |

### Checklist Reels
- [ ] Hook nos primeiros 3 segundos
- [ ] Texto em tela (pouco, grande)
- [ ] Legendas ativadas
- [ ] CTA claro no final
- [ ] Link na bio mencionado
- [ ] Padrão visual consistente
