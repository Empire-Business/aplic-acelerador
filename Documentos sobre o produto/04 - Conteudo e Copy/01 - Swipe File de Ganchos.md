# 📋 Swipe File de Ganchos (Hook Bank)
## Acelerador de Audiência

> **Uso:** Este documento contém ganchos prontos para copiar, adaptar e usar nas suas peças de conteúdo. Cada gancho foi criado considerando a psicologia do público-alvo: empresários que faturam > R$ 400k/ano, vendem conhecimento e já têm time de conteúdo.

---

## 🎯 Como Usar Este Documento

1. **Escolha o tipo de gancho** de acordo com o objetivo da peça
2. **Copie o gancho** e adapte para o seu tom de voz
3. **Use as variações** para testar diferentes ângulos
4. **Observe as recomendações** de plataforma e formato
